# React DevTools

## O que eu preciso lembrar

React DevTools ajuda a inspecionar componentes, props, state e renderizacoes.

## O que olhar

- arvore de componentes;
- props recebidas;
- state atual;
- contexto;
- renders;
- componentes pesados.

## Perguntas

- O componente recebeu a prop certa?
- O state esta certo?
- O componente renderizou de novo?
- O pai esta passando dado errado?

## Pegadinhas

- Procurar erro no filho quando a prop errada vem do pai.
- Nao olhar state atual.
- Confundir problema de dado com problema visual.

## Resumo mental

React DevTools mostra o que o React acha que esta acontecendo.
