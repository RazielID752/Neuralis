# Network Tab

## O que eu preciso lembrar

Network mostra as requisicoes do navegador.

## O que olhar

- URL;
- metodo;
- status code;
- request payload;
- response;
- headers;
- tempo;
- erro de CORS.

## Perguntas

- A chamada aconteceu?
- Foi para a URL certa?
- Enviou body certo?
- O status foi sucesso?
- A resposta veio no formato esperado?

## Pegadinhas

- Debugar componente quando o problema e API.
- Nao olhar response body.
- Confundir erro de CORS com erro de codigo React.

## Resumo mental

Se envolve API, olho Network cedo.
