# CSS

## O que eu preciso lembrar

CSS controla visual e layout.

Ele decide cor, tamanho, espacamento, alinhamento, responsividade e estados visuais.

## Como eu vou estudar CSS

CSS fica menos confuso quando eu entendo a ordem:

1. seletores;
2. cascata e especificidade;
3. box model;
4. display;
5. flexbox;
6. grid;
7. responsividade;
8. estados e variaveis.

## Perguntas que eu sempre devo fazer

- Esse estilo nao aplica por causa de cascata?
- Isso e problema de tamanho, espacamento ou display?
- Preciso de flex ou grid?
- Esse layout funciona no celular?
- O foco do teclado esta visivel?

## Pegadinhas grandes de CSS

- Confundir `margin` e `padding`.
- Esquecer `box-sizing: border-box`.
- Usar `position: absolute` para tudo.
- Brigar com especificidade.
- Fazer layout desktop e esquecer mobile.
- Remover foco visual.

## Resumo mental

CSS e caixa, cascata e layout. Quando entendo esses tres, o resto fica menos misterioso.
