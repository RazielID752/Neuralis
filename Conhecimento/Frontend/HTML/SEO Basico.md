# SEO Basico

## O que eu preciso lembrar

SEO e ajudar buscadores e pessoas a entenderem a pagina.

## Base

```html
<title>Guia de JavaScript</title>
<meta name="description" content="Aprenda JavaScript do basico ao avancado.">
```

## HTML semantico importa

```html
<main>
  <article>
    <h1>Guia de JavaScript</h1>
    <p>Conteudo...</p>
  </article>
</main>
```

## Imagens

```html
<img src="/javascript.png" alt="Logo do JavaScript">
```

## Pegadinhas

- Titulo generico.
- Sem description.
- Varios `h1` sem sentido.
- Imagem importante sem alt.
- Conteudo so via imagem.

## Resumo mental

SEO basico começa com pagina bem estruturada e texto claro.
