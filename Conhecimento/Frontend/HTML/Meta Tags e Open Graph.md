# Meta Tags e Open Graph

## O que eu preciso lembrar

Meta tags descrevem a pagina para navegador, buscadores e redes sociais.

## Tags comuns

```html
<title>Minha pagina</title>
<meta name="description" content="Descricao da pagina">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
```

## Open Graph

```html
<meta property="og:title" content="Minha pagina">
<meta property="og:description" content="Descricao para compartilhamento">
<meta property="og:image" content="https://site.com/og.png">
<meta property="og:type" content="website">
```

## Pegadinhas

- Imagem OG com tamanho ruim.
- Description muito vaga.
- Titulo duplicado em varias paginas.
- Esquecer URL absoluta em imagem compartilhada.

## Resumo mental

Meta tags sao o cartão de visita da pagina fora dela.
