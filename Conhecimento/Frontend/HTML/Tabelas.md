# Tabelas

## O que eu preciso lembrar

Tabela serve para dados tabulares, nao para layout.

## Exemplo

```html
<table>
  <thead>
    <tr>
      <th>Nome</th>
      <th>Email</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>Marcos</td>
      <td>marcos@email.com</td>
    </tr>
  </tbody>
</table>
```

## Quando usar

- relatorios;
- comparacoes;
- dados em linhas e colunas;
- tabelas administrativas.

## Quando nao usar

- layout de pagina;
- alinhar cards;
- estrutura visual sem dado tabular.

## Pegadinhas

- Tabela sem `th`.
- Usar tabela para layout.
- Nao pensar em responsividade.

## Resumo mental

Tabela e para dado que faz sentido em linhas e colunas.
