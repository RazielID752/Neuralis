# Teste de Integracao

## O que eu preciso lembrar

Teste de integracao verifica varias partes funcionando juntas.

## Exemplo

Tela de tarefas:

- busca tarefas;
- mostra loading;
- renderiza lista;
- permite marcar como concluida;
- atualiza UI.

## Bom para

- componentes + hooks;
- tela + API mockada;
- formulario + validacao;
- fluxo com varios componentes.

## Pegadinhas

- Virar teste end-to-end disfarçado.
- Mockar tanto que nao integra nada.
- Ignorar estado de erro.

## Resumo mental

Integracao testa conversa entre partes.
