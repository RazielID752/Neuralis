# Teste End to End

## O que eu preciso lembrar

Teste end-to-end testa o fluxo completo como usuario real.

## Exemplo

Fluxo de login:

1. abrir pagina;
2. digitar email;
3. digitar senha;
4. clicar entrar;
5. ver dashboard.

## Bom para

- fluxos criticos;
- login;
- checkout;
- cadastro;
- acoes principais do produto.

## Pegadinhas

- E2E demais fica lento.
- Pode ser instavel se ambiente nao for controlado.
- Nao substitui testes menores.

## Resumo mental

E2E e teste de caminho principal. Poucos, mas importantes.
