# Arquitetura Frontend

Arquitetura frontend e a forma como organizamos codigo, dados, componentes e responsabilidades.

## Conceitos principais

- Organizacao de pastas
- Separacao de responsabilidades
- Componentes de UI vs componentes de feature
- Client state vs server state
- Tratamento de erros
- Performance
- Testes

## Pergunta guia

Esta organizacao deixa o sistema facil de mudar sem espalhar conhecimento por todo lado?
