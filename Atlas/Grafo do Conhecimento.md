## O que eu preciso lembrar

- link para um mapa;
- link para uma nota anterior;
- link para uma aplicacao pratica;
- link para uma pegadinha ou exemplo.

## Como manter o grafo organizado

Quando criar nota nova:

1. linkar para o mapa da area;
2. linkar para pelo menos uma nota anterior;
3. linkar para um exemplo ou exercicio quando existir;
4. evitar nota solta sem contexto;
5. se a nota conecta duas areas, colocar em "Pontes importantes".
